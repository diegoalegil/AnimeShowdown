# Demo

Tour corto de los flujos principales de AnimeShowdown: home, votacion 1v1, ranking ELO, ficha de personaje, torneo, cartas, fantasy, juegos diarios, perfil y modo TV.

## Flujos Cubiertos

| Flujo | Ruta |
|---|---|
| Home publica | `/` |
| Arena de votacion | `/votar` |
| Ranking ELO | `/ranking` |
| Ficha de personaje | `/personajes/frieren` |
| Torneo en vivo | `/torneos/mha-heroes-vs-villains` |
| Cartas coleccionables | `/cartas` |
| Fantasy Showdown | `/fantasy` |
| Anime Daily Trials | `/games` |
| Perfil privado | `/perfil` |
| Modo TV | `/tv` |

Las capturas fuente viven en [docs/screenshots](screenshots/). Cuando se grabe una nueva version del tour, incluir una apertura de sobre desde resultado del servidor y el draft fantasy semanal antes de juegos diarios. El video se mantiene liviano para que el README y la documentacion de portfolio carguen rapido en GitHub.
