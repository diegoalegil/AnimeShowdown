# Branding

AnimeShowdown publica dos familias de logo:

| Ruta | Uso |
|---|---|
| `frontend/public/logo.svg` | Logo optimizado que carga la cabecera de la app. |
| `frontend/public/logo.webp` | Variante raster para heroes, footer, modo TV y metadatos sociales. |
| `frontend/public/logo/assets/AnimeShowdown-logo.svg` | Fuente vectorial principal para brand kit y materiales externos. |
| `frontend/public/logo/assets/AnimeShowdown-logo-light.svg` | Variante clara para fondos oscuros. |
| `frontend/public/logo/assets/AnimeShowdown-emblem.svg` | Emblema compacto para favicons, avatares y usos cuadrados. |

Los assets de `frontend/public/logo/assets/` son el contrato publico de marca.
No se eliminan aunque la app use versiones optimizadas en rutas cortas.
